# Ace-MU: The Verified SLRTCE Resource Hub

Ace-MU is a centralized platform designed to help Mumbai University 
engineering students access study materials easily.

## Features
- Subject-wise organization
- Semester-wise navigation
- Clean and modern UI
- Fast access to notes & PYQs
